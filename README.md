# Тестовое задание для компании LIIS.

## Сборка проекта:

Сборка происходит с помощью cmake версии 3.22 или выше (для сборки необходимы библиотеки Qt (Core, Network) и Mosquittopp):

1. Склонировать репозиторий  в папку:
```
git clone https://github.com/vkuleshov23/LIIS_test
```

2. запустить скрипт build.sh, который соберет проект в папке build

```
./build.sh
```

## Использование программы:

После успешной сборки, программу можно запустить с помощью скрипта:
```
./run.sh
```
В скрипте вызывается скомпилированный файл из папки build
